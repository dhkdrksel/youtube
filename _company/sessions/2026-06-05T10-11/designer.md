## 🔬 유튜브 시각 자료 디자인 명세서 (Spec Draft v1.0)
**프로젝트:** 살림공학 김빵구 - 곰팡이 제거제 공학적 비교 분석
**타겟 플랫폼:** YouTube (7~9분 내외, 고정보 밀도)
**디자인 목표:** 과학적 신뢰도와 시청자의 불안감(Curiosity & Fear)을 동시에 자극하여 몰입도를 극대화.

### 1. 전반적인 디자인 시스템 및 가이드라인

#### 🎨 컬러 팔레트 (Color Palette)
| 역할 | 색상 코드 (HEX) | 사용 목적 | 설명 |
| :--- | :--- | :--- | :--- |
| **Primary (신뢰)** | `#2E7D32` (진한 녹색 계열) | 메인 텍스트, 성공/긍정적 기능, 배경 강조. | 살림공학의 전문성과 청결함 상징. |
| **Secondary (액션)** | `#00BCD4` (밝은 하늘색) | 흐름도(Flow), 포커스 포인트, 핵심 데이터 연결선. | 과학적 '반응'과 움직임 표현에 적합. |
| **Danger (경고/위험)** | `#D32F2F` (강한 빨간색) | 위험 성분, 부작용 경고 문구, 금기 사항 하이라이트. | 시각적 긴장감 및 주의 환기 극대화. |
| **Background** | `#FAFAFA` (밝은 회백색) | 기본 배경 색상. 깔끔하고 정보에 집중하게 만듦. |

#### 🔡 타이포그래피 (Typography)
*   **메인 폰트:** Pretendard 또는 Noto Sans KR (Sans-serif 계열)
    *   *선택 이유:* 가독성이 매우 높고, 디지털 환경(모션 그래픽)에서 명확하게 구조화된 정보를 전달하는 데 최적임.
*   **사용 규칙:** 모든 제목 및 강조 문구는 **굵게(Bold)** 처리하며, 과학 용어는 항상 *이탤릭체*(Italic)를 병행하여 전문성을 부여합니다.

### 2. 핵심 컴포넌트 상세 명세 (Component Specs)

#### Component A: 성분 비교표 (Ingredient Comparison Matrix)
**[스크립트 출처]:** 본론 1 - 대표적인 성분 세 가지 비교 시점
**[목적]:** 복잡한 데이터를 한눈에 정리하여, '우리가 모르는 정보'를 제공하며 신뢰도를 구축.

| 속성 | 상세 명세 (Spec) | 인터랙션/연출 지침 (Motion Guide) |
| :--- | :--- | :--- |
| **레이아웃** | 3~4열 그리드 구조. 열 제목: 성분명 / 작용 원리(화학식) / 표적 효능 / 주의점. | 모든 정보는 정적인 테이블이 아닌, 질문에 답하듯 섹션별로 **팝업/슬라이딩 인** 되어야 함. |
| **데이터 시각화** | 각 성분명은 아이콘(예: $\text{Cl}^-$ 기호)과 함께 표시. 효능 비교는 단순히 텍스트가 아닌, **게이지 바(Gauge Bar)** 또는 **점수 시스템 (0~100%)**으로 직관화. | 데이터 로딩 시, 그래프가 '차지되는' 애니메이션 효과를 주어 전문적이고 역동적인 느낌을 부여. |
| **강조 요소** | 가장 우수한 성분은 Primary Color (`#2E7D32`)로 배경 하이라이트 처리. 주의점 칸의 위험한 조합은 Danger Color (`#D32F2F`)와 경고 아이콘 ($\triangle$)으로 강조. |

#### Component B: 반응 메커니즘 흐름도 (Reaction Mechanism Flowchart)
**[스크립트 출처]:** 본론 1 - 화학적 작용 원리 설명 시점 (예: 산성 성분이 알칼리성 곰팡이에 작용하는 과정)
**[목적]:** 추상적인 화학 개념을 물리적인 '과정(Process)'으로 치환하여 이해하기 쉽게 만듦.

| 속성 | 상세 명세 (Spec) | 인터랙션/연출 지침 (Motion Guide) |
| :--- | :--- | :--- |
| **구조** | 3단계의 흐름 구조: **[1. 문제물질] $\rightarrow$ [2. 작용제 투입] $\rightarrow$ [3. 반응 및 해결]**. 이 단계가 명확한 화살표로 연결되어야 함. | 시간의 흐름(Time-lapse)을 시각화하듯, 왼쪽에서 오른쪽으로 **순차적 이동 애니메이션**을 적용. (좌→우 움직임 필수). |
| **시각 요소** | 각 물질은 3D 모형 또는 입자 형태($\text{H}_2\text{O}$, $\text{Cl}^-$)로 표현. 반응식 ($\rightarrow$) 기호는 Secondary Color (`#00BCD4`)의 에너지 흐름을 가진 커다란 화살표를 사용. | 물질 간 결합(Bonding)은 '붙어나는' 물리적 효과(Particle Attraction)와 함께 보여주고, 분해(Breaking)되는 과정은 폭발하는 듯한 짧고 강렬한 애니메이션으로 처리. |
| **텍스트** | 반응식의 핵심 화학 용어는 자막으로 크게 노출하며, 해당 부분에만 Primary Color로 포커싱됨. |

#### Component C: 위험 물질 혼합 경고 다이어그램 (Danger Mixing Flowchart)
**[스크립트 출처]:** 본론 2 - 절대 섞으면 안 되는 조합을 설명하는 섹션
**[목적]:** 공포심과 위기감을 자극하여, '프리미엄 가이드북'의 필요성으로 연결시키는 결정적인 시각 자료.

| 속성 | 상세 명세 (Spec) | 인터랙션/연출 지침 (Motion Guide) |
| :--- | :--- | :--- |
| **레이아웃** | 2개 이상의 위험 성분 A와 B가 중앙의 '혼합 용기'를 향해 접근하는 구조. | 가장 먼저 A와 B의 개별적 안정성(Primary Color)을 보여준 후, 두 성분이 근접할 때부터 긴장감을 고조시켜야 함. |
| **연출 기법** | 위험 조합이 발생하면, 배경 전체가 짧게 **빨간색 섬광(Flash)** 효과를 내고, 용기에서 연기/독가스 입자가 폭발하는 듯한 비주얼을 삽입. | 이 섹션의 모든 시각적 에너지는 Danger Color (`#D32F2F`)에 집중되어야 하며, 사운드와 동기화하여 공포감을 극대화해야 함. |
| **결론 제시** | 경고가 최고조에 달했을 때, 화면 하단에 '공학적 원리 기반의 안전 가이드'라는 문구와 함께 Solution Component (Component A/B를 종합한 우리의 솔루션)이 Primary Color로 등장하며 위협을 중화시키는 듯 연출해야 함. |
