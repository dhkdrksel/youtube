#### 🧪 1. Hero Component - 메인 실험 장비 목업 위에 들어갈 타이틀/설명

**[Headline]**
"단순한 감(感)이 아닌, 공학적 검증에 기반한 살림의 원리. [살림공학 김빵구]가 설계합니다."

**[Sub-Copy/설명 캡션]**
"수많은 시행착오 끝에 찾아낸 '감'은 사실 패턴과 법칙의 조합입니다. 저희는 수십 년간 축적된 생활 데이터를 공학적 시스템으로 분석하여, 가장 효율적인 살림의 원리(Optimal Living Protocol)를 도출했습니다. 다음 내용은 실제 데이터 측정 결과와 검증 과정을 거친 보고서 전문을 확인해 주십시오."

#### 📈 2. Graph Component - 그래프/인과관계 다이어그램 주변 설명
*(목적: 시각화된 데이터를 해석해주고, 왜 이 법칙이 작동하는지 논리적으로 설득)*

**[섹션 타이틀]**
"데이터 기반 분석 결과: 시간 대비 효율성을 극대화하는 최적의 동선 경로 (Optimal Workflow Analysis)"

**[본문 설명 캡션]**
"생활 공간에서의 비효율성은 눈에 보이지 않는 '시간 누수'로 발생합니다. 저희가 측정하고 검증한 핵심 지표는 **'동선 중복도 계수(Path Overlap Coefficient)'**입니다. 기존 방식은 여러 단계를 거치며 불필요하게 이동하는 구간이 많아, 에너지와 시간을 낭비했습니다. 아래 그래프에서 확인되듯이, [김빵구의 공학적 솔루션]을 적용했을 때 동선 중복도가 **평균 X% 감소**하며 시간당 효율성이 극적으로 상승했음을 수치로 증명합니다."
